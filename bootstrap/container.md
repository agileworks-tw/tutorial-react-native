# Container

<!--舉例 bootstrap vs pchome24 -->

### container 類別

* fixed-width
* 確保你的頁面會呈現為定寬的 RWD 樣式
* 使用情境：想要頁面以固定寬度出現在螢幕中間

**語法**

```html
<div class="container">
  ...
</div>
```

### container-fluid 類別

* full-width
* 以全寬 RWD 設計呈現你的頁面
* 使用情境：想要頁面總是呈現整個螢幕

**語法**

```html
<div class="container-fluid">
  ...
</div>
```
